## 💻 Using ContextOS for the First Time

When you open ContextOS, you work with simple commands. Here is how to start:

- Type `contextos start` to launch the program.
- Use `contextos add "Your context information"` to add details.
- Use `contextos list` to see saved data.
- Use `contextos help` for more options.

ContextOS keeps track of information you add. This helps AI tools to perform better because they "remember" past details.
